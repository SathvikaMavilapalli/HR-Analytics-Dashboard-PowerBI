Project Overview
This project is an HR Analytics Dashboard created using Power BI. It helps analyze employee data, attrition rate, department-wise performance, gender distribution, average age, and job satisfaction.

Tools Used
- Power BI
- Excel
- DAX
- Data Cleaning
- Data Visualization

Key Insights
- Total employees and attrition count
- Attrition rate by department
- Gender-wise employee distribution
- Average age and tenure analysis
- Job role and salary-based insights
